# Meta Agent Specification

## Purpose
High level controller for strategy, scheduling, and long term decision making.

## Inputs
• historical events  
• trends and analytics  
• performance summaries  

## Outputs
• workflow initiation  
• scheduling signals  
• agent instructions  

## Responsibilities
• determine which workflows to run  
• decide pacing of content cycles  
• coordinate agents at a high level  

## Event Emission
• agent.start  
• agent.success  
• agent.error
