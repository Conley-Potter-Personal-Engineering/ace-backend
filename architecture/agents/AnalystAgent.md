# Analyst Agent Specification

## Purpose
Analyzes performance metrics and determines creative drivers.

## Inputs
• performance_metrics  
• published_posts  
• experiments  
• scripts  
• creative patterns  

## Outputs
• insights for OptimizationAgent  
• updated creative patterns  

## Responsibilities
• analyze experiment performance  
• detect high performing content  
• store notes  
• influence future scripts  

## Event Emission
• analytics.evaluate.start  
• analytics.evaluate.success  
• analytics.evaluate.error
