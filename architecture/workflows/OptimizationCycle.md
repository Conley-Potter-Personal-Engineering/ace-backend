# Optimization Cycle Workflow

## Purpose
Improves creative strategy based on performance and trend data.

## Steps
1. analytics.evaluate.start
2. analyze performance metrics
3. update creative patterns
4. generate optimization instructions
5. analytics.evaluate.success

## Inputs
• experiments  
• performance_metrics  
• creative_patterns  
• trend_snapshots

## Outputs
• updated patterns  
• strategy instructions

## Failure Handling
• fallback to baseline strategy  
• log all errors to system events
